# Documentación de Servidor AsesoriAPP

Se utilizan diagramas de [mermaid.js](https://mermaid.js.org/) cuando se requiera expresar un gráfico.

## Terminología común

- **ITV:** Instituto tecnológico de Veracruz.
- **PADIEER:** Programa de Asesorias De Ingenierías Eléctrica, Electrónica y Renovables.
- **Estudiante:** Persona que estudia en el _ITV_ que utiliza la aplicación de AsesoriAPP, y tiene una cuenta registrada en el servidor. Se puede utilizar como sinónimo de usuario. Pertenece a cualquier carrera que imparte el _ITV_.
- **Asesor:** _Estudiante_ que forma parte de _PADIEER_ como resultado de conocer de ciertas materias, y está capacitado para impartir esos conocimientos a otros _estudiantes_.
- **Asesoría:** Una hora de enseñanza acerca de alguna materia, impartida por un _asesor_ hacia uno o más _estudiantes_, usualmente en el salón de _PADIEER_, aunque es posible que se imparte en otros lugares del instituto. Dentro de la aplicación solo se registran los datos cronológicos de la asesoría, el _asesor_, un _estudiante_ y la materia de la asesoría.
